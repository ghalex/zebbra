A loader can have multiple colors and sizes:

```jsx
<Box>
  <Loader />
  <Loader color='danger' size='large' />
  <Loader color='#FFF000' />
</Box>
<Box>
  <Loader color='blue'>Loading...</Loader>
</Box>
<Box color='success'>
  <Loader color='#FFF000' inverted vertical>Loading...</Loader>
</Box>
```
